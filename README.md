# 🚀 Posting Upload API Service

- 게시글을 비밀번호 설정하여 업로드하는 API 서비스

# 📆 개발 기간
- 2022.09.06 ~ 2022.09.07

# 기능
- 유저가 포스팅 업로드
  - 비밀번호 설정 가능
  - 이모지 포함 기능

- 유저가 올린 포스팅을 비밀번호 입력후 수정/삭제 가능
- Pagination(20개의 포스팅 단위)

# 🛠 기술 스택
- Backend: `Django Rest Framework`
- DB: `MySQL`
- Etc: `Docker`
- Tool : `Git`
